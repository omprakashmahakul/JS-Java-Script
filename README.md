# 🎨 JavaScript Examples

Welcome to a beginner-friendly set of JavaScript samples designed for learning and experimenting.

## 📁 Files Included

- `Js_01.js` - 🔧 Variables with `let`, `const`, and `var`.
- `js_02.js` - 🧠 Data types: number, boolean, string, `undefined`, and `null`.
- `js_03.js` - 🕵️ Type checking using `typeof`.
- `Js_04.js` - 💬 Simple console log example.

## 🚀 How to Use

1. Open the JavaScript files in any text editor.
2. Run them in a browser console or with Node.js.
3. Observe output and modify the code to learn how each concept works.

## 🌟 Why This Helps Beginners

- Builds confidence with core JavaScript syntax.
- Demonstrates how variables and data types work.
- Encourages hands-on practice using console output.
- No setup or dependencies are needed.
